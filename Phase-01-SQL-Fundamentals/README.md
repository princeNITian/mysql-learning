# Phase 1 — SQL Fundamentals

> **Duration:** 1–2 Weeks  
> **Difficulty:** Beginner → Intermediate

---

# Goal

Build a strong foundation in SQL and MySQL by learning how relational databases work, writing SQL queries, and integrating MySQL with Node.js applications.

By the end of this phase, you should be able to build a complete CRUD backend using MySQL.

---

# Prerequisites

- MySQL Server (AWS RDS)
- MySQL Workbench
- Node.js
- mysql2 package
- Basic JavaScript knowledge

---

# Learning Outcomes

After completing this phase, you will be able to:

- Explain what a relational database is.
- Design simple database schemas.
- Create and modify databases and tables.
- Perform CRUD operations.
- Filter and sort data.
- Use aggregate functions.
- Apply SQL constraints.
- Connect Node.js with MySQL.
- Build a REST API backed by MySQL.

- Node.js

- MySQL
- mysql2

- Create User
- Get Users

- Update User
- Delete User
